This is a simple python project to write a asteroid game using python and pygame

step used to Initialize development environment

1. create a new uv project 
   uv init --python 3.13 your-project-name
   cd your-project-name

2. create a virtual envirioment at the top level of the project directory
   uv venv

3. activate the virtual environment
   source .venv/bin/activate

4. add pygame library
   uv add pygame==2.6.1

5. make sure pygame is installed
   uv run -m pygame
